# HabitTracker - Proje Özeti

## Proje Tanımı
HabitTracker, kullanıcıların günlük alışkanlıklarını takip etmelerini sağlayan modern bir web uygulamasıdır. GitHub'ın contribution graph'ına benzer bir takvim görünümü ile kullanıcılar yıl boyunca ilerlemelerini görselleştirebilir.

## Temel Özellikler
1. **Alışkanlık Yönetimi**: Oluşturma, düzenleme, silme
2. **Günlük Takip**: Checkbox ile tamamlama işaretleme
3. **Contribution Takvimi**: GitHub tarzı yıllık görünüm
4. **İstatistikler**: Mevcut seri, en uzun seri, toplam tamamlanan
5. **Zamanlama**: Planlanan saat ve süre belirleme
6. **Tema Desteği**: Koyu/Açık tema + Vurgu rengi seçimi

## Teknik Gereksinimler
- React + TypeScript frontend
- Express.js + SQLite backend
- Tailwind CSS v3 styling
- Vite build tool
- Responsive tasarım (mobil uyumlu)

## Hedef Kullanıcı
Günlük rutinlerini takip etmek ve alışkanlık oluşturmak isteyen bireyler.

## Başarı Kriterleri
- Kolay kullanılabilir arayüz
- Hızlı ve responsive performans
- Görsel olarak motive edici tasarım
- Hem masaüstü hem mobil uyumluluk
